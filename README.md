# Iron Man - Stark Industries Portfolio

![Iron Man UI](public/og-image.png)

A high-performance, cinematic Iron Man-themed portfolio and UI built with **Next.js**, **React**, and **Tailwind CSS**. Designed to feel like J.A.R.V.I.S. is running a system diagnostic on the Mark LXXXV armor.

## Features

- **Cinematic Experience**: Engaging scroll animations and dynamic visual effects.
- **Glassmorphism UI**: High-end modern UI with Stark Industries aesthetics.
- **Performance Optimized**: Built with Next.js 15+ and React 19.
- **Smooth Scrolling**: Implemented using Lenis for a seamless browsing experience.
- **Fully Responsive**: Adapts perfectly to all screen sizes.

## Tech Stack

- **Framework**: Next.js (App Router)
- **UI Library**: React
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Scroll Effects**: Lenis
- **Icons**: Phosphor Icons

## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/rajanvenkat321-sudo/iron-man.git
   cd iron-man
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the application**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
